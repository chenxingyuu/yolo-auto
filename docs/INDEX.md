# Documentation Index

This index is the navigation entrypoint for repository policy and architecture.

## Core

- `README.md` - setup, operations, and MCP usage.
- `docs/QUALITY-GATES.md` - executable quality and merge gates.
- `docs/CAPABILITY_BOUNDARIES.md` - supported scope and explicit non-goals.
- `docs/ITERATIONS.md` - chronological change log for maintainers.

## Workflow

1. Read this index.
2. Confirm policy in `docs/QUALITY-GATES.md`.
3. Run `scripts/validate-repo.sh` before PR creation.
